# wenku8-download2026

轻小说文库（wenku8）通用下载脚本。

介绍

苦于一直没找到好用、稳定、适配 2026 年当前站点结构的 PC 端轻小说文库下载器，于是整理优化了一版通用爬取脚本，可直接在浏览器运行，无需安装软件、无需配置环境。


使用说明

1. 打开任意轻小说文库小说目录页

2. 按 F12 进入开发者工具 → 切换到 Console 控制台

3. 粘贴脚本代码并回车

4. 等待自动下载完成，浏览器会自动导出 TXT 文件

详细演示：

https://www.bilibili.com/opus/1184533022299389959

适用范围

- 支持文库站内绝大多数轻小说

- Windows / Mac 通用

- 仅支持 Chrome / Edge 等现代浏览器

- 仅供个人学习、备份使用，请勿商用或传播
